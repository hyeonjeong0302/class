table 만들기

html에서 표를 구현할 때

  • tr(table row): 행(가로)
  • td(table data): 열(세로, 행의 내용)
  • th(table heading): 행, 열의 제목(기본이 굵음체, 가운데 정렬)
  • 속성 colspan="2": 2칸의 열을 병합
  • 속성 rowspan="2": 2칸의 행을 병합
